## ✨ Features

- 🏎️ **Blazing Fast** — Built with Vite 8 for instant HMR and optimized production builds
- 🎨 **Modern Dark UI** — Sleek dark theme with purple/pink gradient accents and glassmorphism effects
- 📱 **Fully Responsive** — Mobile-first design that looks stunning on all screen sizes
- 🎬 **Rich Animations** — Scroll-reveal animations powered by Framer Motion
- 🧭 **Client-Side Routing** — Seamless navigation with React Router v7
- 🎯 **SEO-Friendly** — Semantic HTML, meta tags, and optimized structure
- 📝 **Multi-Page Architecture** — Home, About, Services, Contact, Projects, Reviews, Blog
- 🔄 **Logo Marquee** — Animated scrolling brand/service ticker
- ❓ **Interactive FAQ** — Accordion-style FAQ section
- 📊 **Stats & Metrics** — Animated counters and data visualization sections
- 🎴 **Blog Cards** — Custom cursor interaction on blog grid
- 🎪 **Case Studies Portfolio** — Visual showcase of client work
- 📬 **Consultation Form** — Lead capture form with validation

## 🚀 Installation

### Prerequisites

- [Node.js](https://nodejs.org/) v18.0.0 or higher
- [npm](https://www.npmjs.com/) v9.0.0 or higher (or yarn/pnpm)

### Setup Steps

```bash
# 1. Clone the repository
git clone https://github.com/H0NEP0T-466/ktechnologypro-task.git

# 2. Navigate to the project directory
cd ktechnologypro-task

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server with HMR |
| `npm run build` | Create optimized production build |
| `npm run preview` | Preview production build locally |
| `npm run lint` | Run ESLint on all source files |

## ⚡ Usage

### Development

```bash
npm run dev
```

Starts the Vite development server with Hot Module Replacement (HMR). Edit any file in `src/` and see changes instantly in the browser.

### Production Build

```bash
npm run run build
```

Outputs an optimized, minified build to the `dist/` directory, ready for deployment to any static hosting platform (Vercel, Netlify, Cloudflare Pages, etc.).

### Linting

```bash
npm run lint
```

Runs ESLint with the configured React hooks and React refresh plugins to ensure code quality.
